# Face Analysis Engine 👤

Real-time face analysis system built with OpenCV and Keras that detects **age, gender, and emotion** from webcam feed.

Optimized to run smoothly on low-spec hardware (my old HP14s laptop) without lag.

---

## ✨ Features
- Real-time webcam detection
- Age estimation
- Gender classification
- Emotion recognition (Happy, Sad, Angry, Neutral, etc.)
- Custom coordinate tracking loop for better performance on weak CPU
- FPS counter + confidence display
- Screenshot capture

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Keras / TensorFlow
- Pre-trained models (Haar Cascade + CNN)

---

## 🚀 How to Run

```bash
git clone https://github.com/boikzdev/face-analysis-engine.git
cd face-analysis-engine
pip install -r requirements.txt
python main.py
```
